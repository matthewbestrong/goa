Heading Example with Merriweather

This is an example of the body text using Open Sans. Notice how the clean and modern feel of the sans-serif font works well with the classic and formal style of the Merriweather serif font in the headings.

This is a blockquote using a combination of both fonts.

By combining a serif font for headings and a sans-serif font for body text, we can create a visually appealing and readable layout for different types of content. This pairing helps establish a contrast while maintaining harmony on the page.
